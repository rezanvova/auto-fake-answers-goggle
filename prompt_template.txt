Тебе нужно сгенерировать веса (вероятности) для каждого варианта ответа на вопросы ниже.
Веса должны создавать логичное распределение ответов, имитирующее реальных респондентов.

ВАЖНО:
- Веса должны быть от 0.0 до 1.0
- Сумма весов для каждого вопроса НЕ обязательно должна быть равна 1.0
- Для радио-кнопок (один вариант) сделай так, чтобы один вариант был популярнее остальных
- Для чекбоксов (несколько вариантов) используй веса чтобы показать популярность каждого варианта
- Большие веса = более частые ответы

### Вопрос N: [Текст вопроса]

Ответы:

- Вариант ответа 2: 0.XX

ПРАВИЛА ФОРМАТИРОВАНИЯ (ОБЯЗАТЕЛЬНЫЕ):
1. Каждый вопрос НАЧИНАЕТСЯ с "### Вопрос N:" где N - номер вопроса
2. Следующая строка ОБЯЗАТЕЛЬНО "Тип: radio" или "Тип: checkbox" (только эти два варианта!)
3. Пустая строка
4. Строка "Ответы:" (с двоеточием!)
5. Каждый ответ начинается с "- " (дефис и пробел)
6. Формат ответа: "- Текст ответа: 0.XX" (двоеточие и пробел перед числом)
7. Текст ответа ДОЛЖЕН ТОЧНО совпадать с текстом в форме (копируй как есть!)
8. Веса ТОЛЬКО в формате 0.XX (например: 0.60, 0.35, 0.05)
9. Между вопросами - ОДНА пустая строка

- Никаких дополнительных объяснений
- Никаких комментариев
